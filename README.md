# Forma API

## Instructions

1. Create a virtual environment
```bash
python -m venv .venv
```
or

```bash
python3 -m venv .venv
```

2. Activate virtual environment
In MacOS/Linux/Unix:
```bash
source .venv/bin/activate
````

3. Install dependencies
```bash
make install
```

4. Run tests
```
make test
```
